# SCT_ML_3 - Decision Tree Classifier

## Objective
Build a Decision Tree Classifier to predict whether a customer will purchase a product or service using the Bank Marketing Dataset.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Google Colab

## Machine Learning Workflow
1. Load dataset
2. Data preprocessing
3. Encode categorical values
4. Split dataset
5. Train Decision Tree model
6. Evaluate model performance
7. Visualize Decision Tree

## Outputs
- Accuracy Score
- Classification Report
- Decision Tree Visualization
